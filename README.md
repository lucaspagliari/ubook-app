# Ubook App
App in production: [link](ubook-app.vercel.app)

## Steps to run 
### NPM 
```
npm install
npm run dev
```
```
npm run test:unit
```

### YARN 
```
yarn install
yarn dev
```

```
yarn test:unit
```

### Compiles and minifies for production
```
npm run build
yarn build
```

### Technologies used:
- Vue 3 + Typescript
- TailwindCSS
- Jest
